## ✔ 주요 기능

- 🏬 가맹점관리
  - 가맹점 등록
  - 가맹점 경고 기능
  - 폐점한 매장 업데이트 및 조회

- 🍽 메뉴관리
  - 메뉴 등록
  - 메뉴 판매 상태 변경 및 조회(판매/미판매)
  - 필요한 재고 발주하기

- 💻 주문관리
  - 가맹점 별 (당일) 주문 내역 확인(영수증)
  - 테이블 별 주문 내역 확인
  - 손님 식사 완료 여부에 따라 주문 상태 변경

- 📈 매출관리
  - 당일 & 이번 달 매출 조회
  - 당일 & 이번 달 결제 건수 조회

- 📋 테이블 오더
  - 테이블 등록 & 비활성화
  - 메뉴 확인 & 장바구니 담기
  - 장바구니 담은 메뉴 주문하기
  - 해당 테이블 현재 주문 내역 확인
  - 홀짝 게임
